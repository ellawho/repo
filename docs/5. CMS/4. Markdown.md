### Heading

```
# H1 - Heading
## H2 - Heading
### H3 - Heading
```

### Bold 

```
**bold text**
```

### Italic

```
*italicized text*
```

### Blockquote

```
> blockquote
```
### Ordered List

```
1. First item
2. Second item
3. Third item
```

### Unordered List

```
- First item
- Second item
- Third item
```

### Code

```
`code`
```

### Horizontal Rule

```

```
