## Field Alias
* Using `whatever-name` with `:` in front of `siteMetadata`,  `whatever-name` will become the alias. 

    ```
    query MyQuery {
    site {
        info:siteMetadata {
        title
        }
    }
    }
    ```

## Query Keyword, Name and Gatsby Clean
* `MyQuery` is the Query Keyword, it requires to be unique 
  
    ```
    query MyQuery {
    site {
        info:siteMetadata {
        title
        }
    }
    }
    ```
